# Asteroid Simulator Dashboard ☄️📊

An interactive 3D Orbital Simulator and Analytical Dashboard designed to map, filter, and analyze space objects using the "NASA Near-Earth Asteroids & Close Approaches" dataset from Kaggle.

## 🚀 Project Overview
This project bridges the gap between Data Engineering and Frontend Web Development. It features a complete pipeline from raw data ingestion to a high-performance, real-time 3D visualization embedded inside an analytical dashboard.

## 🛠️ Tech Stack
* **Data Pipeline:** Microsoft Fabric (Lakehouse ingestion & data cleansing)
* **Backend & Analytics Dashboard:** Python + Streamlit (Advanced filtering, KPIs, and main UI)
* **3D Engine:** Three.js + OrbitControls (WebGL rendering, interactive camera, and dynamic telemetry HUD)
